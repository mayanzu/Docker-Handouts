# 讲义模板使用指南

## 1. 模板概述

本讲义模板是一个基于HTML、CSS和JavaScript的静态网页模板，专为课堂教学和讲座演示设计。模板采用红色主题，PPT风格布局，支持响应式设计，适合在不同设备上显示。

### 核心特性
- **红色主题**：唯一使用红色作为强调色，保持视觉一致性
- **大字体设计**：基础字体大小为1.5rem（150%），适合投影显示
- **自动分页**：内容超出页面时自动创建新页面
- **动态字体调整**：根据内容量自动调整字体大小
- **响应式布局**：适配不同屏幕尺寸
- **简洁导航**：支持上一页/下一页按钮、键盘导航和触摸滑动

## 4. CSS样式设置

### 4.1 字体系统

模板使用CSS变量定义字体大小，实现动态调整：

```css
--base-font-size: 1.5rem;          /* 基础字体大小（150%） */
--body-font-size: calc(var(--base-font-size) * 1.0625); /* 正文字体 */
--heading-font-size-1: calc(var(--base-font-size) * 1.75); /* 一级标题 */
--heading-font-size-2: calc(var(--base-font-size) * 2.5);  /* 二级标题 */
--heading-font-size-3: calc(var(--base-font-size) * 3.5);  /* 三级标题 */
--small-font-size: calc(var(--base-font-size) * 0.875);    /* 小字体 */
--tiny-font-size: calc(var(--base-font-size) * 0.75);      /* 极小字体 */
```

### 4.2 间距系统

使用相对单位定义间距，确保与字体大小保持一致：

```css
--spacing-xs: calc(var(--base-font-size) * 0.375);  /* 极小间距 */
--spacing-sm: calc(var(--base-font-size) * 0.5);    /* 小间距 */
--spacing-md: calc(var(--base-font-size) * 0.75);   /* 中等间距 */
--spacing-lg: calc(var(--base-font-size) * 1.125);  /* 大间距 */
--spacing-xl: calc(var(--base-font-size) * 1.5);    /* 超大间距 */
```

### 4.3 颜色系统

模板严格使用红色作为唯一强调色：

```css
--color-accent: #dc2626;         /* 主强调色（红色） */

--color-accent-dark: #b91c1c;    /* 深红色（ hover 状态） */
```

## 5. JavaScript功能说明

### 5.1 核心功能

1. **页面导航**：
   - 支持上一页/下一页按钮点击
   - 支持键盘方向键和空格键导航
   - 支持触摸滑动导航

2. **动态字体调整**：
   - 根据页面内容量自动调整字体大小
   - 内容少时增大字体，内容多时减小字体
   - 字体大小范围：1.2rem - 2.0rem

3. **自动分页**：
   - 检测页面内容是否溢出
   - 溢出时自动创建新页面
   - 保留原始章节序号格式

4. **页码管理**：
   - 自动更新总页数
   - 实时显示当前页码
   - 禁用边界页的导航按钮

## 7. 注意事项

### 7.1 内容组织

1. **合理分段**：每个页面内容不宜过多，建议控制在10-15行文本以内
2. **模块使用**：根据内容类型选择合适的模块，保持页面整洁
3. **序号格式**：章节标题使用 `Part X.X` 格式，内容页使用阿拉伯数字

### 7.2 技术细节

1. **自动分页**：当内容溢出时，系统会自动创建新页面，新页面会继承当前页面的样式
2. **字体调整**：字体大小会在1.2rem-2.0rem之间自动调整，确保内容完整显示
3. **响应式设计**：在小屏幕设备上，多列布局会自动切换为单列
4. **导航限制**：第一页时上一页按钮禁用，最后一页时下一页按钮禁用

### 7.3 常见问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 页面内容溢出 | 内容过多或字体过大 | 系统会自动创建新页面，或手动分割内容 |
| 字体大小不合适 | 内容量与字体大小不匹配 | 系统会自动调整，或手动修改基础字体大小 |
| 导航按钮不工作 | 页码计算错误 | 确保 `totalPages` 变量正确更新，可刷新页面重新计算 |
| 样式显示异常 | CSS文件路径错误 | 确保CSS文件与HTML文件在同一目录下 |
| 响应式布局失效 | 屏幕尺寸检测错误 | 检查CSS媒体查询设置，确保语法正确 |

### 7.4 性能优化

1. **图片使用**：如需添加图片，建议压缩图片大小，避免影响加载速度
2. **代码精简**：删除不需要的内容模块和样式，减少文件大小
3. **浏览器缓存**：在服务器环境下，启用浏览器缓存提高加载速度
4. **字体选择**：使用系统默认字体或Web安全字体，避免加载外部字体文件
